# Overview

This repository contains the complete solution for the Linux System Administration laboratory assignment. The objective of this assignment is to demonstrate practical knowledge of Linux commands, system administration concepts, file system management, user permissions, storage analysis, and input/output operations.

Every question has been performed on an Ubuntu Linux system using the Bash shell. Commands were executed directly from the terminal, outputs were verified, screenshots were captured, and detailed technical explanations have been provided.

The repository has been organized according to the assignment guidelines and includes reports, commands, outputs, observations, and screenshots for every task.

# Key Observations

- Ubuntu uses the Bash shell by default.
- Linux permissions play an important role in protecting data.
- Hard links share the same inode as the original file.
- Symbolic links only reference the original file path.
- File descriptors are essential for Linux I/O operations.
- Output and error streams can be redirected independently.
- Low inode utilization indicates a healthy file system.
- Disk usage analysis helps prevent storage-related failures.
